Control Algorithm

The system implements intelligent load shedding based on power consumption thresholds. When power exceeds the set threshold, loads are automatically turned OFF according to priority levels.

Threshold Control Logic

Threshold Range (Watts) Action Taken Loads Turned OFF Remaining Power Status
≥ 140 Monitor Only None 140W Safe Operation
120 - 139 Turn OFF Load 2B Feeder 2 - Load 2B (20W) 120W Warning
100 - 119 Turn OFF Feeder 2 Both Feeder 2 loads (40W) 100W Alert
50 - 99 Turn OFF Feeder 2 + Load 1B Feeder 2 (40W) + Feeder 1 - Load 1B (50W) 50W Critical
< 50 Emergency Shutdown All loads except Load 1A 50W Emergency

Logic Explanation

The system follows a priority-based load shedding approach:

  • Load Priority Order: Load 2B → Load 2A → Load 1B → Load 1A
  • Feeder Priority: Feeder 2 (Low Priority) loads are turned OFF first
  • Threshold Ranges: System responds differently based on threshold value
  • Auto Recovery: When power drops below threshold, system restores loads in reverse order
